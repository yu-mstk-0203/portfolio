# AI業務改善家ポートフォリオ

GitHub Pagesで公開するための静的ポートフォリオサイトです。

## 方針

- 居住地は「九州在住」に留める。
- 過去勤務や関係先の詳細は公開しない。
- 実績は匿名ケースとして扱う。
- 公開する柱は「生成AIパスポート」「市役所との実務実績」「記事執筆」「予約システム構築」。

## 公開方法

### GitHub Actionsを使う場合

1. GitHubに公開リポジトリを作成する。
2. このフォルダの内容をpushする。
3. リポジトリの `Settings > Pages` で `GitHub Actions` を選ぶ。
4. `Deploy static portfolio to GitHub Pages` の実行完了後、PagesのURLを確認する。

### ブランチから直接公開する場合

1. GitHubに公開リポジトリを作成する。
2. このフォルダの内容をpushする。
3. `Settings > Pages` で `Deploy from a branch` を選ぶ。
4. 対象ブランチと `/root` を選ぶ。

## 差し替えが必要なもの

- `contact/index.html` の連絡先。
- 必要に応じて、GitHubプロフィールURL。
- 独自ドメインを使う場合は、GitHub Pages設定とDNS設定。

## 公開前チェック

- 詳しすぎる居住地が入っていないか。
- 過去勤務や関係先の詳細が推測できる表現がないか。
- 未許諾の固有名詞や非公開情報が入っていないか。
- 連絡先として公開してよいURLやメールだけが載っているか。
